## Quick Start

### Option 1: Use the Icon Generator
See the main `PWA_SETUP_README.md` for the icon generator tool.

### Option 2: Create Manually

1. Create a 512x512px PNG with your logo
2. Use these recommended settings:
   - Background: #7B61FF (cosmic purple)
   - Logo/Symbol: Centered with 20% padding
   - Format: PNG with transparency
   - Rounded corners: Optional (looks modern)

3. Export/resize to all required sizes

### Option 3: Use Online Tools

- **PWABuilder**: https://www.pwabuilder.com/
- **Favicon Generator**: https://realfavicongenerator.net/
- **PWA Icon Generator**: https://progressier.com/pwa-icons-generator
